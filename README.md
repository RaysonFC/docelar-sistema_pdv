# docelar-sistema_pdv# 🍫 Docelar — Sistema de Caixa

Sistema de caixa para a doceria Docelar.

## Arquivos
- `index.html` — Tela principal do caixa
- `style.css` — Estilização
- `script.js` — Lógica do sistema
- `relatorio.html` — Relatório de vendas

## Como usar
Baixe os 4 arquivos e abra o `index.html` no navegador.
Mantenha todos os arquivos na mesma pasta.
